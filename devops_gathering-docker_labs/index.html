Docker Labs at Bochum

Dear Friends of Docker!

Inspire the people: though the first edition of DevOps Gathering Conference hasn’t even started yet, we are preparing the first round – Docker Birthday Talks in Bochum in March. But we can offer you today some interesting 23.3.2017 half-day labs. Prepare yourself with useful information and tips from the Docker experts.

  • Docker on ARM
    • Speaker: Dieter Reuter && Niclas Mietz
    • Overview of Docker on ARM
    • Check your docker tools for ARM
    • Build Docker Images on ARM
    • Setup an IoT Project on ARM - Blink!
  • Docker for Windows
    • Speaker: Dieter Reuter && Stefan Scherer
    • Setup Windows Docker Container at Windows Server 2016
    • Size your images
    • Package windows service into a container
    • Hack Around with windows container
    • Review at a real example
  • Docker Orchestration with swarming mode
    • Speaker: Peter Rossbach
    • Learn Swarming Mode Basics
    • Setup a Swarm Cluster
    • Prepare your Images
    • Rolling your services forward
    • Network, Port Publishing and Loadbalancing
    • Service Discovery
    • Orchestration
  • Docker for Java Developers
    • Speaker: Roland Huss
    • Create your java images with Maven
    • Use an IDE with docker
    • Setup integration test
    • Optimize your java base images
    • How you can run your container with resource constraints?
  • Prepare your docker images with Gitlab/CI
    • Speaker: Niclas Mietz
    • Setup your own GitLab/CI for your team
    • Create your own docker images producing pipeline
    • Test and validate your containers
    • Create and use runners
    • Build good docker images

We wish you a peaceful Docker winter! We hope you are be(e) prepared for spring birthday party …
