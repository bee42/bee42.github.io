DevOps Gathering - Container Workshops

Docker Workshops in Bochum - 2017/3/23

Dear DevOps Friends!

One week after the Docker Birthday Week, we invite you to continue the celebrations with us. Within our DevOps Gathering Conference 2017, we bring together Container experts, beginners and interesting individuals. During interesting half-day workshops you will have the chance to practice DevOps tools to improve your knowledge and become an expert.

Docker on ARM & Orchestration

Speaker: Dieter Reuter* and Peter Rossbach

  • Check your docker tools for ARM
  • Build Docker Images on ARM
  • Setup an IoT Project on ARM
  • Learn Swarming Mode Basics
  • Setup a Swarm Cluster
  • Prepare your Images
  • Rolling your services forward
  • Network, Port Publishing and Loadbalancing
  • Service Discovery
  • Orchestration

Docker for Windows

Speaker: Dieter Reuter* and Stefan Scherer

  • Setup Windows Docker Container at Windows Server 2016
  • Size your images
  • Package windows service into a container
  • Hack Around with windows container
  • Review at a real example

Docker for Java Developers

Speaker: Roland Huss

  • Create your java images with Maven
  • Use an IDE with docker
  • Setup integration test
  • Optimize your java base images
  • How you can run your container with resource constraints?

Prepare your docker images with Gitlab/CI

Speaker: Niclas Mietz

  • Setup your own GitLab/CI for your team
  • Create your own docker images producing pipeline
  • Test and validate your containers
  • Create and use runners
  • Build good docker images

“Bootiful” Containers with Spring-Boot and Docker

Speaker: Kevin Wittek

* Run Spring-Boot Apps inside a Docker container
* Build your Spring-Boot Docker image using Gradle
* Make Spring-Boot Apps configurable with Environment-Variables
* Unit- and Integration-Test your Spring-Boot App with Spock and Docker
* Deploy and configure Spring-Boot microservices using docker-compose

*Docker Captain
